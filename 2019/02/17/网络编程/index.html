网络编程

本文最后更新于：2021年3月8日 晚上

一，网络编程的三要素

1.1 IP地址

IP（Internet Protocol Address）地址是物理地址，用于标识网络中的计算机设备

查看本机IP地址：cmd->ipconfig

查看网络是否连通：cmd->ping 域名/IP地址

IPv4：是一个32的二进制数，表示形式：xxx.xxx.xxx.xxx，能够表示42亿多个IP地址

IPv6：表示形式：xxxx:xxxx:xxxx:xxxx:xxxx:xxxx:xxxx:xxxx，每16个字节为一组，分成8组，每一个x可以表示0~f

注：本机的IP地址：localhost、127.0.0.1

1.2 端口号

端口号（port）是逻辑地址，用于标识设备中的应用的

端口号的范围：065535，特殊的一般如01023之间的端口号用于一些知名（系统）的网络服务和应用

表示形式：ip 地址 : 端口号

1.3 传输协议

数据在网络中进行传输时需要遵循的规则

在 java.net 包中定义了使用 UDP、TCP 协议完成数据传输的类

UDP：User Datagram Protocol，用户数据报协议

​ 特点：

  1. 无需建立连接，发送端不会去确认接收端是否存在

  2. 速度快

  3. 不可靠

  4. 应用领域：在线视频

  5. 代码中：传输的数据与目标地址要封装在同一个数据包对象中

核心类：

  1. DatagramSocket：封装了数据的收发功能
  2. DatagramPacket：数据包，封装要传输的数据，如果是发送方还要封装发送的目标地址

TCP：Transmission Control Protocol，传输控制协议

​ 特点：

  1. 在底层通过三次握手来建立连接，保证数据的安全
    1. 第一次，客户端向服务器发出连接请求，等待服务器的确认
    2. 第二次，服务器向客户端发送一个响应，通知客户端收到了连接的请求
    3. 第三次，客户端再次向服务器发送确认信息，来确认连接
  2. 速度慢
  3. 可靠的
  4. 应用领域：下载

核心类：

  1. ServerSocket：表示服务端
  2. Socket：表示客户端

二，InetAddress

概念：InetAddress 类是 java.net 包下的一个用于获取域名和IP地址的类

方法：该类的方法都是静态方法

  1. getByName(String host)
  2. getLocalHost()
  3. getAllByName(String host)

三，UDP的使用

3.1 一个端接收一个端发送

// 发送端
public class UDPClient1 {
	public static void main(String[] args) throws Exception {
		// 当前这个类只用于发送，不接收数据，因此不需要明确自己的端口号
		DatagramSocket ds = new DatagramSocket();
		String msg = "闭嘴";
		byte[] b = msg.getBytes();
        /
        * 参数：
        * 1.要发送的数据，类型的字节数组
        * 2.发送数据的大小
        * 3.InetAddress对象，该对象中要封装目标的IP地址
        * 4.端口号，端口号的值要和接收端的端口号一致
        */
		DatagramPacket dp = new DatagramPacket(b, b.length, InetAddress.getByName("10.11.53.114"), 1234);
        // 发送功能，因为你在发送，所以你的DatagramPacket中要有目标的IP地址及端口号
		ds.send(dp);
        // 释放资源
		ds.close();
	}
}
// 接收端
public class UDPClient2 {
	public static void main(String[] args) throws Exception{
		// 当前这个类用于接收数据，要设置自己的端口号，才能让发送者在发送时能明确目标地址
		DatagramSocket ds = new DatagramSocket(1234);
		byte[] b = new byte[1024];
        // 作为接收端中的byte数组是作为缓冲使用的
		DatagramPacket dp = new DatagramPacket(b, b.length);
        // 接收功能，当程序运行到这句话时会处于阻塞状态，等待发送端的发送动作，当发送端执行了发送功能，数据包中的byte数组会接收到数据
		ds.receive(dp);
        // 获取byte数组中的数据
		byte[] msg = dp.getData();
        // 获取接收到的有效数据大小
		int len = dp.getLength();
		System.out.println(new String(msg,0,len));
	}
}

四，TCP的使用

4.1 服务端收，客户端发

// 服务端（收）
public class TCPServer {
	public static void main(String[] args) throws Exception{
        	// 创建服务器对象，并设置当前服务器的端口号，才能让发送者在发送时能明确目标地址
		ServerSocket ss = new ServerSocket(1234);
		// 使当前服务器处于阻塞状态，等待其他客户端的连入，返回连入的客户端对象
		Socket s = ss.accept();
		// 根据连入的客户端对象s获取字节输入流，该输入流用于接收连入的客户端所发来的数据
		InputStream is = s.getInputStream();
		byte b[] = new byte[1024];
		int len = is.read(b);
		System.out.println("接收到了："+new String(b,0,len));
		is.close();
		s.close();
		ss.close();
	}
}
// 客户端（发）
public class TCPClient {
	public static void main(String[] args) throws Exception{
        	// 创建客户端对象，并明确所要连入服务器的ip地址和端口号，如果要连入的服务器已经执行到了accept()方法，那么此时，当前这个客户端对象就连入了指定的服务器，服务器也接收到了连入的客户端对象
		Socket s = new Socket("10.11.53.114", 1234);
       	// 获取客户端的输出流对象，该输出流用于向已经连入的服务器进行数据的发送
		OutputStream os = s.getOutputStream();
		os.write("你好".getBytes());
		os.close();
		s.close();
	}
}

4.3 聊天室

// 服务端
public class TCPServer {
	public static List<Socket> sockets = null;
	public static void main(String[] args)  throws Exception{
		ServerSocket ss = new ServerSocket(6969);
		sockets = new ArrayList<Socket>();
		new Thread(new Runnable() {
			@Override
			public void run() {
				while(true){
					try {
						Socket s = ss.accept();
						sendAll(s);
						sockets.add(s);
						new Thread(new Runnable() {
							@Override
							public void run() {
								while(true){
									try {
										InputStream is = s.getInputStream();
										byte[] b = new byte[1024];
										int len = is.read(b);
										String msg = new String(b,0,len);
										sendAll(s,msg);
									} catch (Exception e) {
									}
								}
							}
						}).start();
					} catch (Exception e) {
					}
				}
			}
		}).start();
	}
	
	public static void sendAll(Socket s, String str) throws Exception{
		String ip = s.getInetAddress().getHostAddress();
		String time = new SimpleDateFormat("HH:ss:mm").format(new Date());
		String msg = ip+"在"+time+"发送了："+str;
		System.out.println(msg);
		BufferedWriter bw = new BufferedWriter(new FileWriter("C:\\Users\\qianfeng\\Desktop\\log.txt",true));
		bw.write(msg);
		bw.newLine();
		bw.flush();
		for (Socket socket : sockets) {
			OutputStream os = socket.getOutputStream();
			os.write(msg.getBytes());
		}
	}
	
	public static void sendAll(Socket s) throws Exception{
		String ip = s.getInetAddress().getHostAddress();
		String time = new SimpleDateFormat("HH:ss:mm").format(new Date());
		String msg = ip+"在"+time+"加入了聊天室";
		System.out.println(msg);
		for (Socket socket : sockets) {
			OutputStream os = socket.getOutputStream();
			os.write(msg.getBytes());
		}
	}
}

4.4 文件上传

// 客户端
public class TCPClient {
	public static void main(String[] args) throws Exception {
		/*
		 * 将本地文件上传至服务器
		 * 1.读取本地文件
		 * 2.上传到服务器
		 * 3.读取服务器的回写数据
		 * 
		 * 准备工作
		 * 1.数据源：要上传的文件 E:\test
		 * 
		 * 步骤：
		 * 1.创建本地的FileInputStream，来明确要读取的本地文件
		 * 2.创建Socket对象，绑定要连入的服务器的ip和端口号
		 * 3.使用Socket的getOutputStream获取网络输出流对象
		 * 4.使用本地的FileInputStream的read()方法读取本地文件
		 * 5.使用Socket的网络输出流的write()方法将读取到的流对象写到服务器
		 * 6.使用Socket的getInputStream获取网络输入流对象
		 * 7.使用Socket的网络输入流的read()方法读取服务器回写的数据
		 * 8.释放资源
		 */
		
		FileInputStream fis = new FileInputStream("E:\\test\\黑寡妇.png");
		Socket s = new Socket(InetAddress.getByName("10.11.53.114"), 5678);
		OutputStream os = s.getOutputStream();
		int len = 0;
		byte[] b = new byte[1024];
		while((len = fis.read(b)) != -1){
			os.write(b, 0, len);
		}
		/*
		 * bug：图片已经成功上传到服务器了，但是客户端没有接收到回写的数据
		 * 原因：不是客户端没有接收到回写的数据，而是服务器没有执行到回写功能
		 * 我们需要在图片完毕后，告知服务器，告知的方式是向服务器发送一个结束的标记
		 */
		s.shutdownOutput();

		InputStream is = s.getInputStream();
		while((len = is.read(b)) != -1){
			System.out.println(new String(b,0,len));
		}
		s.close();
		fis.close();
	}
}
// 服务端
public class TCPServer {
	public static void main(String[] args) throws Exception {
		/*
		 * 将本地文件上传至服务器
		 * 1.读取客户端上传的文件
		 * 2.将文件保存到本地服务器
		 * 3.回写数据
		 * 
	 	 * 准备工作
		 * 2.明确目标地址: E:\资料
		 * 
		 * 步骤：
		 * 1.创建ServerSocket对象，指定端口号
		 * 2.通过ServerSocket的accept()方法，获取客户端对象
		 * 3.使用客户端对象的getInputStream获取网络输入流对象
		 * 4.判断“同学们的好图”这个目录是否存在，不存在，则创建
		 * 5.创建本地的FileOutputStream，明确读取到的文件往哪里存
		 * 6.使用客户端的网络输入流对象的read()方法，读取文件
		 * 7.使用本地输出流对象的write()方法写文件
		 * 8.使用客户端对象的getOutputStream获取网络输出流对象
		 * 9.使用网络输出流对象的writer()方法回写“上传成功”
		 * 10.释放资源
		 * 
		 */
		ServerSocket ss = new ServerSocket(5678);
		while(true){
			Socket s = ss.accept();
			new Thread(){
				public void run() {
					try {
						InputStream is = s.getInputStream();
						File f = new File("E:\\资料\\同学们的好图");
						if(!f.exists()){
							f.mkdirs();
						}
						int len = 0;
						byte[] b = new byte[1024];
						String filename = new SimpleDateFormat("yyyyMMddHHssmm").format(new Date())+""+new Random().nextInt(1000000000)+".jpg";
						FileOutputStream fos = new FileOutputStream(new File(f,filename));
						while((len = is.read(b)) != -1){
							fos.write(b, 0, len);
						}
						OutputStream os = s.getOutputStream();
						os.write("上传成功！！！".getBytes());
						fos.close();
						s.close();
					} catch (Exception e) {
					}
				}
			}.start();
		}
	}
}
